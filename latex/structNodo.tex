\doxysubsection{Detailed Description}
\doxylink{structNodo}{Nodo} del trie 4-\/ario para almacenar genes. 

Cada nodo representa una base del gen (A, C, G, T). En los nodos hoja se guarda una lista con las posiciones donde aparece el gen correspondiente.
